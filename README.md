# Customer Registry

## Full Stack MERN Project

**Developed by:** Tharunya Palagani

## Project Overview

Customer Registry is a Full Stack MERN application developed as part of the APSCHE Full Stack Developer (MERN Stack) Virtual Internship. The application enables users to manage customer information and complaints through a simple and responsive web interface.

## Features

- Add new customers
- View all customers
- Update customer details
- Delete customer records
- Register customer complaints
- View complaint history
- Edit complaint details
- Dashboard with customer and complaint statistics
- Responsive user interface

## Technology Stack

### Frontend
- React.js
- Vite
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Deployment
- Frontend: Vercel
- Backend: Render

## Installation

### Clone Repository

```bash
git clone https://github.com/tharunyapalagani/Customer-Registry.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm start
```
